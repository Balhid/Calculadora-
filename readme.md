Aplicación de Consola en JavaScript

## Descripción
Este proyecto corresponde al Módulo 4 del curso Fundamentos de Programación en JavaScript.  
Consiste en una aplicación interactiva que se ejecuta en la consola del navegador y permite realizar operaciones matemáticas básicas.

## Funcionalidades
- Menú interactivo mediante prompt()
- Operaciones matemáticas: suma, resta, multiplicación y división
- Validación de entradas del usuario
- Uso de funciones para modularizar el código
- Uso de objetos para encapsular operaciones
- Uso de arreglos para almacenar historial
- Uso de condicionales y bucles

## Tecnologías utilizadas
- JavaScript (ES6)
- Consola del navegador

## Cómo ejecutar
1. Abrir el navegador
2. Abrir la consola (F12)
3. Copiar y pegar el código del archivo app.js
4. Presionar Enter
